# Data Cleaning & Preprocessing
**Dataset:** Titanic Dataset  
**Tools:** Python, Pandas, Numpy, Seaborn, Matplotlib  

## Problem Statement
Clean and prepare raw Titanic data for further analysis.

## Approach
- Handled missing values (Age → median, Embarked → mode, Cabin → dropped)
- Removed duplicate rows
- Converted data types to category where needed
- Renamed columns for better readability

## Results
- Clean dataset with 891 rows and 0 missing values
- Saved as titanic_cleaned.csv
